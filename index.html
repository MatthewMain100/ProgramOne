  • git clone "(http)path to repository"
  • git status - статус локального репозитория
  • git add path to file - добавить новый файл под контроль ГИТА
  • git commit -am"added new file" - зафиксировать изменения и дать им коммент
  • git push - отправить все закоменченные изменения на репозиторий
  • git pull - забрать все свежие обновления
